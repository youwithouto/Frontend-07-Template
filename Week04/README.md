## Algorithms on Strings

- Trie
  - 大量高重复字符串的存储于分析

- KMP

- Wildcard
  - 带通配符的字符串模式
- Regular Expression
  - 字符串通用模式匹配
- State Machine
  - 通用字符串分析
- LL & LR
  - 字符串多层级结构分析
